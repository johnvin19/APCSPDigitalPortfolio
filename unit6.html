6.6 - Cryptography Basics



1. Caesar Cipher - (A) Shifts letters forward over a certain number(1-25). For example, A becomes B and B becomes C, and so forth. (B) Shifts letters backwards over a certain number(1-25). (C) Can easily be broken

Simple Substitution Cipher - (A) Uses keyword to start, then fills the rest of the alphabet. For example, keyword zebra, ZEBRACDFGHIJKLMNOPQSTUVWXY. (B) Have normal list of alphabets and encrypted list connect and write messages. (C) Harder to break but can be broken with frequency analysis.

Vigenere Cipher - (A) Uses a vignere chart and keyword, connect keyword letters to normal alphabet to get encrypted. For example, if message is Sixty eight and keyword is monkey, the message would repeat letters like monkeymonk. (B) Use keyword and backtrack the letters. (C) Hard to break but can be broken with frequency analysis.
